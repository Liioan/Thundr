Note app created using [T3 stack](https://create.t3.gg)

## hotkeys

- global
  - `ctrl+j`: change theme
- note creator
  - `shift+n`: open note creator
  - `escape`: close note creator
- markdown note
  - `ctrl+e`: render/edit note
